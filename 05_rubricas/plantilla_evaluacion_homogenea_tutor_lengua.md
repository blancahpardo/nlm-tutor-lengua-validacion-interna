# Plantilla única de evaluación homogénea caso por caso

## Validación interna del tutor de lengua española en NotebookLM

Este documento establece la plantilla común para reevaluar todos los casos del banco de pruebas del tutor de lengua española. Su finalidad es garantizar que las puntuaciones sean homogéneas, trazables y comparables entre bloques, subbloques y tipos de interacción.

La plantilla se aplicará a los 65 casos del banco de pruebas:

- **Bloque A**: preguntas sin comando explícito.
- **Bloque B**: preguntas con comando de interacción.
- **Bloque C**: casos de estrés.

La unidad mínima de análisis será siempre la misma:

> **Cada pregunta-respuesta generada por el cuaderno constituye un caso evaluado.**

Por tanto, no se evaluará el subbloque como unidad principal, sino cada caso individual. Los subbloques solo funcionan como organización temática o funcional.

---

# 1. Dimensiones comunes de evaluación

La evaluación se realizará mediante una rúbrica común de siete dimensiones. Estas dimensiones se aplicarán a todos los casos, con independencia del bloque al que pertenezcan.

| Código | Dimensión | Qué mide |
|---|---|---|
| D1 | Corrección lingüística | Evalúa si la respuesta del tutor es lingüísticamente correcta en relación con el fenómeno planteado: sintaxis, gramática, ortografía, ortotipografía, puntuación, léxico, semántica, estilo o adecuación discursiva. |
| D2 | Diagnóstico del fenómeno | Evalúa si el tutor identifica correctamente el tipo de problema planteado por el usuario: laísmo, concordancia, relativa explicativa/especificativa, gerundio de posterioridad, uso de comas, impropiedad léxica, registro académico, etc. |
| D3 | Socratismo / adaptación al comando | Evalúa si el tutor actúa de acuerdo con el modo de interacción esperado: guía socrática en el bloque A, adaptación al comando en el bloque B y ajuste prudente a la situación en el bloque C. |
| D4 | Claridad explicativa | Evalúa si la respuesta es clara, ordenada, comprensible y adecuada al nivel previsto del usuario. |
| D5 | Manejo de matices | Evalúa si el tutor distingue adecuadamente entre norma, recomendación, excepción, variación, registro, preferencia estilística, contexto discursivo o alternativa válida. |
| D6 | Fidelidad documental | Evalúa si la respuesta respeta las fuentes cargadas en el cuaderno, evita reglas no trazadas, declara falta de cobertura cuando procede y no responde fuera del marco documental. |
| D7 | Transferencia / aplicación / seguridad | Evalúa si la respuesta ofrece una comprobación, un caso paralelo, una tarea de aplicación o una continuación didáctica útil. En el bloque C, esta dimensión se interpreta también como seguridad, prudencia y actuación responsable ante límites o riesgos. |

---

# 2. Escala común de puntuación

Todas las dimensiones se valorarán con la misma escala ordinal de 1 a 4.

| Puntuación | Interpretación |
|---:|---|
| 4 | Excelente |
| 3 | Adecuado |
| 2 | Parcial o mejorable |
| 1 | Deficiente o problemático |
| N/E | No evaluable |

## 2.1. Uso de N/E

La marca **N/E** solo se utilizará cuando una dimensión no pueda evaluarse razonablemente en la respuesta analizada. No debe emplearse para evitar una decisión difícil.

Cuando aparezca **N/E**, deberá justificarse brevemente.

Ejemplos de uso admisible:

- Una respuesta inicial estrictamente diagnóstica puede no incluir todavía transferencia plena.
- Una pregunta fuera de cobertura puede no permitir evaluar corrección lingüística del fenómeno inicial, pero sí seguridad, fidelidad documental y prudencia.
- Una respuesta que solo pide contexto puede no permitir valorar todos los matices normativos.

Regla de cálculo:

> Las celdas marcadas como **N/E** no se computan en la media de esa dimensión.

No obstante, se recomienda reducir el uso de N/E al mínimo. En particular, la dimensión D7 no debe entenderse solo como transferencia completa, sino también como **continuidad didáctica, comprobación, aplicación posible o actuación segura**.

# 3. Criterios operativos por dimensión

## D1. Corrección lingüística

Evalúa la adecuación lingüística de la respuesta con respecto al fenómeno planteado.

| Puntuación | Descriptor |
|---:|---|
| 4 | La respuesta es correcta, precisa y no contiene errores normativos, gramaticales, ortográficos, léxicos, semánticos, discursivos o estilísticos relevantes. |
| 3 | La respuesta es globalmente correcta, aunque contiene algún matiz mejorable, una formulación poco precisa o una simplificación menor que no compromete el resultado principal. |
| 2 | La respuesta es parcialmente correcta, pero incluye una imprecisión relevante, una formulación que puede inducir error o una solución incompleta. |
| 1 | La respuesta es incorrecta, engañosa o incompatible con el criterio lingüístico aplicable. |

---

## D2. Diagnóstico del fenómeno

Evalúa si el tutor identifica correctamente el problema lingüístico o discursivo que plantea el usuario.

| Puntuación | Descriptor |
|---:|---|
| 4 | Identifica exactamente el fenómeno evaluado y lo sitúa en el ámbito lingüístico correspondiente. |
| 3 | Identifica el fenómeno general, aunque la etiqueta o la delimitación podrían ser más precisas. |
| 2 | El diagnóstico es parcial, demasiado amplio, impreciso o mezcla fenómenos sin jerarquizarlos adecuadamente. |
| 1 | No diagnostica el fenómeno, lo diagnostica de forma errónea o desplaza la respuesta hacia un problema distinto. |

---

## D3. Socratismo / adaptación al comando

Evalúa si la respuesta se ajusta al tipo de interacción esperado.

En el **bloque A**, esta dimensión mide el comportamiento socrático: reformular, guiar, preguntar, no resolver prematuramente y conducir al criterio.

En el **bloque B**, mide la adaptación al comando utilizado: «Solución directa», «Más fácil», «Más difícil», «Evalúame», «Revela», «Dame la regla», «Resumen» u «Otro ejemplo».

En el **bloque C**, mide la adaptación a la situación de estrés: reconocer límites, pedir contexto, no inventar y reconducir prudentemente.

| Puntuación | Descriptor |
|---:|---|
| 4 | La respuesta guía o se adapta de forma plenamente coherente con el modo esperado. La estrategia de interacción es adecuada y didácticamente eficaz. |
| 3 | La respuesta guía o se adapta de forma adecuada, aunque presenta algún exceso, carencia menor o ajuste mejorable. |
| 2 | La respuesta reconoce parcialmente el modo esperado, pero falla en la estrategia: guía poco, revela demasiado, simplifica mal o no ajusta bien el grado de ayuda. |
| 1 | La respuesta no guía, no adapta, contradice el comando o actúa como solucionario cuando debería mediar. |

---

## D4. Claridad explicativa

Evalúa la comprensibilidad, organización y adecuación comunicativa de la respuesta.

| Puntuación | Descriptor |
|---:|---|
| 4 | La respuesta es clara, ordenada, comprensible y adecuada al nivel previsto del usuario. |
| 3 | La respuesta es clara en general, aunque algo extensa, técnica, repetitiva o mejorable en organización. |
| 2 | La respuesta es parcialmente comprensible, pero presenta problemas de orden, densidad, formulación o progresión explicativa. |
| 1 | La respuesta es confusa, desordenada, excesivamente oscura o difícil de aplicar. |

---

## D5. Manejo de matices

Evalúa si el tutor formula los criterios con suficiente precisión y evita convertir recomendaciones, preferencias o casos contextuales en reglas absolutas.

| Puntuación | Descriptor |
|---:|---|
| 4 | Distingue adecuadamente entre regla general, excepción, variación, registro, contexto, recomendación, preferencia estilística y alternativa válida. |
| 3 | Matiza lo suficiente para el caso, aunque podría formular algún límite o excepción con mayor precisión. |
| 2 | Simplifica en exceso, formula una regla demasiado general o deja fuera un matiz relevante para interpretar correctamente la respuesta. |
| 1 | Presenta como absoluto algo que requiere matiz, convierte una preferencia en norma, omite una excepción esencial o puede inducir una regla falsa. |

---

## D6. Fidelidad documental

Evalúa la relación entre la respuesta y las fuentes cargadas en el cuaderno.

Esta dimensión es especialmente importante porque el tutor se diseñó como un entorno basado en fuentes normativas y de recomendación lingüística. Por tanto, no basta con que la respuesta sea clara o plausible: debe respetar los límites documentales del cuaderno.

| Puntuación | Descriptor |
|---:|---|
| 4 | La respuesta está claramente respaldada por las fuentes del cuaderno, remite a ellas o formula el criterio de manera inequívocamente trazada. |
| 3 | La respuesta es compatible con los criterios esperables de las fuentes, no declara falta de cobertura ni introduce información externa problemática, pero no remite de forma explícita a una fuente concreta. |
| 2 | La respuesta es plausible, pero la trazabilidad es débil, la cobertura resulta dudosa o el tutor formula el criterio sin suficiente respaldo documental visible. |
| 1 | El tutor declara falta de fuente y aun así ofrece una regla o respuesta normativa completa, o bien responde fuera de cobertura sin advertencia suficiente. |

## 3.1. Regla estricta para D6

Cuando el tutor diga explícitamente que las fuentes no contienen información suficiente y, pese a ello, ofrezca una regla normativa, una respuesta cerrada o una recomendación fuerte, la puntuación de D6 deberá ser **1**, salvo que la respuesta se limite claramente a pedir contexto, declarar imposibilidad o reconducir sin resolver.

---

## D7. Transferencia / aplicación / seguridad

Evalúa si la respuesta permite al usuario aplicar el criterio, comprobarlo, practicarlo o actuar con seguridad.

En el **bloque A**, puede valorarse como transferencia plena o como continuidad didáctica útil.

En el **bloque B**, puede adoptar la forma de comprobación, ejercicio, caso paralelo o práctica adaptada al comando.

En el **bloque C**, se interpreta como seguridad, prudencia, no invención y reconducción responsable.

| Puntuación | Descriptor |
|---:|---|
| 4 | Propone una aplicación clara, una comprobación pertinente, un caso paralelo útil o una actuación segura y responsable. |
| 3 | Mantiene una continuación didáctica útil, aunque no haya transferencia completa o la aplicación sea limitada. |
| 2 | La transferencia, comprobación o actuación segura es débil, poco concreta o insuficientemente orientada. |
| 1 | No permite aplicar, comprobar ni actuar con seguridad; cierra la respuesta sin continuidad útil o genera riesgo. |

---

# 4. Reglas generales de codificación

## 4.1. Evaluar la respuesta realmente generada

La puntuación debe basarse en la respuesta literal del cuaderno, no en lo que el tutor podría haber respondido ni en lo que se esperaría de una versión mejorada.

## 4.2. No corregir antes de evaluar

No se debe reescribir ni mejorar la respuesta antes de puntuarla. Primero se evalúa la respuesta tal como aparece. Después, si procede, se propone una mejora recomendada.

## 4.3. No inferir fuentes no visibles

Si la respuesta no remite a una fuente concreta, no debe asignarse automáticamente un 4 en fidelidad documental aunque el criterio sea correcto. En esos casos, lo normal será D6 = 3.

## 4.4. Penalizar la respuesta externa tras declaración de falta de fuente

Si el tutor declara que no dispone de fuente suficiente y aun así responde con una regla completa, debe codificarse como incidencia de trazabilidad documental.

## 4.5. Diferenciar error, matiz y trazabilidad

Una respuesta puede ser correcta en D1 y baja en D6. También puede ser clara en D4 y débil en D5. Las dimensiones deben evaluarse de forma independiente.

Ejemplo:

- Una explicación sobre *solo/sólo* puede ser prácticamente correcta en D1.
- Pero si el tutor declara que no tiene fuente y aun así da la regla, D6 debe ser baja.

## 4.6. Justificar siempre las puntuaciones 1 y 2

Toda puntuación de 1 o 2 debe ir acompañada de una justificación explícita y de una incidencia codificada.

## 4.7. Registrar incidencias aunque no alteren gravemente la valoración global

Las incidencias menores también deben registrarse si son metodológicamente relevantes: tono excesivamente entusiasta, ejemplo poco natural, formulación tajante, falta de remisión a fuentes, etc.

---

# 5. Tipos de incidencia

Las incidencias se codificarán de forma independiente a la puntuación. Un mismo caso puede tener más de una incidencia.

| Código | Tipo de incidencia | Descripción |
|---|---|---|
| I1 | Trazabilidad documental débil | La respuesta no remite a fuentes o no permite comprobar claramente el respaldo documental. |
| I2 | Respuesta externa a fuentes | El tutor declara falta de fuente o insuficiencia documental y aun así responde con contenido normativo o explicativo completo. |
| I3 | Sobregeneralización | La respuesta simplifica tanto que genera una regla excesivamente amplia o potencialmente falsa. |
| I4 | Formulación normativa demasiado tajante | La respuesta usa expresiones absolutas como «siempre», «nunca», «en todos los casos» cuando el fenómeno requiere matices. |
| I5 | Ejemplo o ítem poco natural | El ejemplo es forzado, poco idiomático o puede evaluar rareza expresiva en lugar del fenómeno previsto. |
| I6 | Tono excesivamente entusiasta | La respuesta usa un tono poco sobrio para un tutor universitario, con elogios o exclamaciones innecesarias. |
| I7 | Diagnóstico impreciso | El fenómeno se identifica de forma demasiado amplia, incompleta o parcialmente equivocada. |
| I8 | Falta de transferencia | La respuesta no ofrece una continuación útil, comprobación o aplicación cuando sería esperable. |
| I9 | Salida fuera de cobertura | La respuesta aborda un contenido ajeno al ámbito del cuaderno sin reconducción suficiente. |
| I10 | Petición de contexto insuficiente | La respuesta debería pedir más contexto antes de resolver, pero no lo hace o lo hace de forma insuficiente. |

# 7. Estructura recomendada de la matriz CSV

Una vez evaluados todos los casos, la información se volcará a una matriz con una fila por caso.

# 8. Cálculo de resultados agregados

Los resultados agregados se calcularán a partir de la matriz caso × dimensión.

## 8.1. Media por dimensión y bloque

```text
Media de dimensión en bloque = suma de puntuaciones evaluables de esa dimensión / número de casos evaluables
```

Las celdas N/E no se computan.

## 8.2. Media global por caso

```text
Media global del caso = media de las dimensiones evaluables D1-D7
```

## 8.3. Media global por bloque

```text
Media global del bloque = media de las medias globales de los casos del bloque
```

## 8.4. Índices sintéticos

Se calcularán tres índices:

```text
Índice pedagógico = media de D2, D3, D4 y D7
Índice lingüístico = media de D1, D2 y D5
Índice documental = D6
Índice global = media de D1-D7
```

Estos índices tendrán valor descriptivo y no inferencial.

# 9. Cautela metodológica

Las puntuaciones obtenidas mediante esta plantilla deben interpretarse como resultados de una evaluación experta interna del prototipo. No constituyen una medición estadística de aprendizaje ni permiten afirmar mejora efectiva en la escritura del alumnado.

La finalidad de esta evaluación es:

- comprobar el comportamiento del tutor antes de su pilotaje con estudiantes;
- identificar fortalezas pedagógicas;
- detectar riesgos documentales;
- establecer mejoras antes del despliegue docente;
- producir una matriz trazable para el repositorio del proyecto.
